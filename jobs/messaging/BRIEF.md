## Structure

For each landing section, in order, provide:
- **Section name + job** (what this section must accomplish for the visitor)
- **Headline** (final copy, not placeholder)
- **Supporting copy** (final copy)
- **Proof elements** (what evidence/data/visual backs the claim)
- **Mechanism note** (what interaction or dynamic element carries this section — scroll-driven curve, carousel, live module, etc. The site must feel like a pitch-deck-demo, not a brochure)
- **CTA if any**

7. Footer
